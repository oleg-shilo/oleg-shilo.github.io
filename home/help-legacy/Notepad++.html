Notepad++

Notepad++ is an Open Source source code editor with syntax highlighting. It is very light and at the same time quite powerful. It can be very handy when editing scripts on the system where no development tools are installed  (see Notepad++ home page  for details). 

Notepad++ comes with the plugin dedicated for working with CS-Script. You can find more details on the pluginn home page: CSScript.Npp.
